Q&A Lite

Simple demo app backed by AWS API Gateway, Lambda & DynamoDB.

Create User

Start by creating a user. The generated userId will be reused when you ask or answer questions.

Ask a Question

Use your userId to post a new question. It will appear in the Latest Questions list on the right.

Tip: create a user first so the form auto-fills your userId.

Answer a Question

Select a question from Latest Questions (it will auto-fill the Question ID), then write an answer.
